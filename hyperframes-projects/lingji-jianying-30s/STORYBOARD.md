# 灵机剪影 30 秒产品视频分镜

**规格**

- 时长：30 秒
- 画幅：1920 x 1080
- 制作：HyperFrames HTML + GSAP
- 图片：APIMart `gpt-image-2`
- 口播：MiniMax TTS
- BGM：MiniMax 音乐生成或 MiniMax 生成能力不可用时的本地兜底

**节奏**

1. 0-6s：痛点开场。多任务碎片飞入，强调来回切换。
2. 6-12s：产品亮相。灵机剪影作为本地桌面工作台出现。
3. 12-18s：素材到文稿。抖音导入、自动转录、写稿审稿。
4. 18-24s：视觉生产。MiniMax 口播字幕、封面、信息卡、动画卡。
5. 24-30s：导出收口。时间线合成后导出 MP4，品牌主张落版。

**画面资产**

- `assets/cover-product.png`：AI 生成主视觉，展示本地桌面工作台。
- `assets/workflow-visual.png`：AI 生成工作流图景，展示从素材到成片。
- `assets/export-scene.png`：AI 生成导出完成场景。
- `assets/*.png`：来自当前系统 `pics/` 的真实界面截图。

**口播对应画面**

- “素材、文稿、配音、字幕和画面来回切换” → 碎片化卡片与真实截图快速堆叠。
- “本地桌面工作台” → 主视觉大屏与产品名称。
- “导入抖音素材，自动转录成文稿” → 欢迎页、导入弹窗、写稿工作台。
- “写稿、审稿、MiniMax 口播和字幕” → 审稿截图、TTS 设置截图。
- “视觉节奏、封面、信息卡、动画卡片” → 编辑器与 AI 助手截图。
- “一键导出 MP4” → 导出弹窗与成片路径。
